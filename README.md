# Real Estate Listings Platform

A full-stack real estate listings application built as part of the **TechKraft Inc. Full Stack Engineer take-home assessment**.

The application provides:

- A public property search experience with filtering and pagination
- A property detail page
- JWT-based authentication
- Role-aware access for admin vs normal users
- Admin-only visibility of internal listing metadata on the listing detail view

This project is implemented using:

- **Backend:** Node.js, Express, TypeScript, PostgreSQL, Prisma ORM
- **Frontend:** React, TypeScript, Vite, Tailwind CSS, shadcn/ui
